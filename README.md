# E-commerce Customer Journey & ROI Analysis

This project analyses cosmetics e-commerce user behaviour to understand conversion drop-off, purchase revenue concentration and growth opportunities.

## What it does
- Loads and cleans 1M raw event records
- Builds a View → Cart → Purchase conversion funnel
- Calculates session-level conversion rates
- Computes top-brand purchase revenue and revenue share
- Generates stakeholder-ready visualisations and business insights

## Key findings
- 202,599 view sessions
- 83,965 cart sessions
- 6,353 purchase sessions
- 3.14% overall purchase conversion
- Top-10 brand revenue: $77,062
- Leading brand share: 30.8%

## Tech stack
Python, Pandas, NumPy, Matplotlib, Seaborn
